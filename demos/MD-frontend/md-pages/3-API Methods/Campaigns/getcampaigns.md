# Campaigns

#


## Get campaigns 

	[Campaigns?site={siteid}&session={sessionkey}]

 returns all campaigns for a specific site ID. 

+ Method

	**GET** (`application/json; utf-8`)

+ Parameters

	+ siteID (Char `13`, `string`)
	+ sessionkey (Varchar `32`, `string`)
	
	
#

+ Model

	```
	{"username":"testuser"}
	```

+ Response 200 (`application/json`)

	+ List < M3Campaign >
