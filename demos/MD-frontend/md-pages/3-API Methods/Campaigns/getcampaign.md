## Get campaign 

	[/{campaignid}?session={sessionkey}]

returns the data fields of a specific campaign.  

+ Method

	**GET** (`application/json; utf-8`)

+ Parameters

	+ campaignID (Char `13`, `string`)
	+ sessionkey (Varchar `32`, `string`)


+ Response 200 (`application/json`)
